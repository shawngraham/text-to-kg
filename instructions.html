Instructions

To modify this application, please follow the steps below:

  1. See what models are available from ollama.ai.
  2. At the command line use the ollama run command with the model name you'd like to employ. For example: 
    ollama run --model tinyllama
  3. Open both the app.py and index.html file in a text editor and modify the relevant lines to point to the model you wish to use. Look for lines similar to: 
    model: "tinyllama"
    and replace "tinyllama" with the name of the model you want to use.
  4. If you wish to modify the system prompts used for generating ontology and CSV, inspect the app.py file and update the prompts as needed. Look for lines similar to: 
    const systemPrompt = "You are a knowledge representation assistant..."

Once you have completed these steps, you can start using the application to upload PDFs, extract text using OCR, and extract entities and relationships as json, or represent them as .gexf.
