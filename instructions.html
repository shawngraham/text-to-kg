Instructions

To use this application, please follow the steps below:

  1. Download Ollama from ollama.ai.
  2. Once downloaded, run Ollama using the command line. Use the ollama run command with the model name you'd like to employ. For example: 
    ollama run --model tinyllama
  3. Open the index.html file in a text editor and modify the relevant lines to point to the model you wish to use. Look for lines similar to: 
    model: "tinyllama"
    and replace "tinyllama" with the name of the model you want to use.
  4. If you wish to modify the system prompts used for generating ontology and CSV, inspect the index.html file and update the prompts as needed. Look for lines similar to: 
    const systemPrompt = "You are a knowledge representation assistant..."

Once you have completed these steps, you can start using the application to upload PDFs, extract text using OCR, and generate ontology and CSV files using the Ollama model of your choice.

If you encounter any issues or have questions, please refer to the Ollama documentation or seek support from their website.
